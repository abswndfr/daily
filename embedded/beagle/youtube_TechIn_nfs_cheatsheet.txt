1. installing TFTP server:

  sudo apt install tftpd-hpa
  sudo systemctl status tftpd-hpa
  
  
2. configuring tftp server:

  the drfeault configuration file of tftpd-hpa server is /etc/default/tftpd-hpa. To configure the tftp server, we need to modify this configuration file and restart the tftpd-hpa service.
  
  
  For modification:

	sudo nano /etc/default/tftpd-hpa	
	sudo chown tftp:tftp /tftp path	
	sudo systemctl restart tftpd-hpa
	
  Installing NFS server

	sudo apt install nfs-kernel-server
	
  Create shared NFS directory
  
	sudo mkdir -p /srv/nfs/TechIn
	sudo chown -R nobody:nogrouup /srv/nfs/TechIn
	sudo chmod 777 /srv/nfs/TechIn	
	
  grant access to the client system for accessing the NFS server
  
	sudo nano /etc/exports	
	sudo exportfs -arv
	sudo systemctl restart nfs-kernel-server
